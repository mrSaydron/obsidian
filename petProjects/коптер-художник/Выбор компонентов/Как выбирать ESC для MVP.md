# Как выбирать ESC для MVP

Источник: [[Комплектующие MVP]], [[Выбор компонентов/Как выбирать моторы для MVP|Как выбирать моторы для MVP]], [[Расчёт веса и тяги MVP]]

## Что такое ESC

`ESC` — Electronic Speed Controller, электронный регулятор скорости.

В коптере ESC стоит между батареей и мотором:

`батарея → ESC → мотор → винт`

Flight controller не питает моторы напрямую. Он только отправляет ESC управляющий сигнал: какую тягу нужно дать каждому мотору. ESC уже сам коммутирует ток из батареи в три фазы бесщёточного мотора.

Для квадрокоптера нужно 4 ESC: по одному на каждый мотор. Они могут быть отдельными модулями или одной платой `4-in-1 ESC`.

## Главный принцип

ESC выбирается не отдельно, а под конкретную силовую связку:

`мотор → винт → батарея → максимальный ток → ESC → охлаждение → разъёмы/проводка`

Главный риск: поставить большой винт или высокий шаг, получить ток выше возможностей ESC и сжечь регулятор в полёте.

## Основные параметры ESC

### Ток

В карточках обычно указывают:

- continuous current — длительный ток;
- burst current — кратковременный пиковый ток.

Ориентироваться нужно в первую очередь на длительный ток. Burst — это не рабочий режим, а короткий запас на пики.

Практическое правило:

| Максимальный ток мотора по thrust table | Минимальный ESC | Лучше для MVP |
|---:|---:|---:|
| 15–20 A | 25–30 A | 30–40 A |
| 25–30 A | 35–40 A | 40–50 A |
| 35–40 A | 50 A | 50–60 A |
| 45–50 A | 60 A | 60–80 A |

Если thrust table показывает максимум около `30 A`, ESC на `40 A` выглядит нормально. Если максимум около `40 A`, лучше смотреть `50–60 A`.

### Напряжение

ESC должен поддерживать напряжение батареи.

Для нашего MVP базовый ориентир — `6S`.

Важно помнить:

- `6S nominal` = 22.2 В;
- полностью заряженная 6S батарея = 25.2 В.

Если ESC указан как `2–4S`, он не подходит для 6S. Нужно искать ESC с поддержкой `6S`, например `3–6S`, `2–6S` или выше.

### Совместимость с мотором и винтом

ESC не выбирается по KV мотора напрямую. Его выбирают по реальному току, который мотор потребляет с конкретным винтом и батареей.

Один и тот же мотор:

- с меньшим винтом может потреблять 15–20 A;
- с большим винтом может потреблять 35–45 A;
- на 4S и 6S будет вести себя по-разному.

Поэтому ключевой документ — thrust table мотора. Там нужно смотреть строку для нужного винта и 6S.

### Управляющий сигнал

Flight controller управляет ESC через сигнальный протокол.

Частые варианты:

- PWM — старый и максимально совместимый вариант;
- OneShot / MultiShot — промежуточные старые протоколы;
- DShot — цифровой протокол, часто используется в современных сборках;
- CAN ESC — более продвинутый вариант, обычно для дорогих/промышленных систем.

Для первого MVP главное:

- чтобы Pixhawk 6C Mini / ArduPilot мог управлять выбранным ESC;
- чтобы ESC поддерживал понятный режим работы с ArduPilot;
- чтобы была документация по калибровке throttle range, если используется PWM.

Для простой сборки под ArduPilot нормально начинать с PWM-совместимых ESC. DShot тоже возможен, но не должен быть причиной выбора, если остальная силовая часть не подтверждена.

### Прошивка ESC

У ESC тоже есть своя firmware.

Частые варианты:

- BLHeli_S / BLHeli_32 — распространены в FPV;
- SimonK — старый вариант;
- проприетарная firmware HobbyWing / T-Motor и других производителей;
- UAVCAN/CAN firmware для smart ESC.

Для нашего MVP важнее не название firmware, а практические вещи:

- стабильная работа на нужном токе;
- поддержка 6S;
- нормальная совместимость с ArduPilot/Pixhawk;
- предсказуемый старт мотора;
- отсутствие случайных desync на больших винтах;
- понятная документация.

### BEC

Некоторые ESC имеют `BEC` — встроенный стабилизатор 5 В для питания электроники.

Для Pixhawk-сборки лучше не полагаться на BEC из ESC как на основное питание flight controller. Нормальная схема:

`батарея → power module → Pixhawk`

ESC в этом случае питают только моторы, а Pixhawk получает питание и измерение батареи через power module.

Если у ESC есть BEC, нужно внимательно понять, надо ли отключать красный провод питания на signal cable. Несколько BEC, соединённых параллельно без понимания схемы, могут создать проблемы.

### Отдельные ESC или 4-in-1

Отдельные ESC:

- проще охлаждать на лучах рамы;
- легче заменить один сгоревший регулятор;
- удобнее для X500/F550/F600-класса;
- больше проводов и пайки.

4-in-1 ESC:

- компактнее;
- меньше проводов в центре рамы;
- удобен для FPV-сборок;
- при поломке одного канала часто меняется вся плата;
- может хуже охлаждаться в тяжёлом hover-коптере.

Для коптера-художника и X500-класса практичнее отдельные ESC на лучах, особенно на первом MVP.

### Охлаждение и размещение

ESC греется, когда через него идёт большой ток.

Проверить:

- есть ли поток воздуха вокруг ESC;
- не закрыт ли ESC термоусадкой/креплением слишком плотно;
- не расположен ли ESC вплотную к flight controller, GPS/compass или слабосигнальным проводам;
- достаточно ли сечение силовых проводов;
- не перегреваются ли разъёмы.

На hover-коптере ESC может долго работать в одном режиме. Поэтому важно смотреть не только короткий максимальный ток, но и температуру после нескольких минут висения.

### Размер, масса и проводка

Для каждого ESC нужно учесть:

- массу;
- длину проводов до мотора и PDB/power distribution;
- тип разъёмов;
- сечение силовых проводов;
- способ крепления к лучу;
- защиту от вибраций и случайного касания пропеллера.

Слишком тонкие провода или слабые разъёмы могут стать узким местом даже при хорошем ESC.

## Текущий выбор

### EMAX BLHeli 40A / 50A 2-6S ESC

Ссылка: https://aliexpress.ru/item/1005009019606904.html

Статус: текущий выбранный вариант для MVP, потому что его проще купить на AliExpress.

Плюсы:

- доступен на AliExpress;
- есть варианты 40A / 50A / 60A / 80A;
- версии 40A+ в этой серии рассчитаны на 2-6S;
- отдельный ESC удобно ставить на лучи X500;
- обычное PWM-управление должно быть совместимо с Pixhawk 6C Mini / ArduPilot;
- по EMAX BLHeli manual версия 40A UBEC имеет 40A continuous и 50A burst, версия 50A UBEC имеет 50A continuous и 60A burst.

Минусы и риски:

- карточка AliExpress оформлена неаккуратно: в заголовке встречаются слова про bidirectional/waterproof/underwater propeller, хотя изображения похожи на обычную EMAX BLHeli ESC-серию;
- нужно проверить, что продавец отправляет именно выбранную версию 40A/50A 2-6S;
- это более бюджетный выбор, чем HobbyWing XRotor Pro;
- встроенный BEC не должен быть основным питанием Pixhawk.

Что проверить перед покупкой:

- выбрать SKU не младше `40A`;
- точную версию ESC и поддержку `2-6S`;
- continuous current и burst current;
- есть ли BEC или это OPTO-версия;
- поддерживаемый управляющий сигнал;
- совместимость с ArduPilot/Pixhawk через PWM/DShot;
- массу одного ESC;
- размеры и способ крепления;
- реальные отзывы по работе с большими винтами.

## Как подобрать ESC для текущих моторов

### Если остаётся Cosmicrc 4108 380KV

По найденной таблице Cosmicrc 4108 380KV даёт ток около `11.8–12.3 A` в найденных тестах, но таблица неполная и не закрывает все нужные винты.

Инженерно:

- `30 A` может хватить по найденной таблице, но покупать его для MVP не стоит;
- `40 A` — базовый выбранный вариант с запасом;
- `50 A` — разумный вариант, если цена/масса отличаются не критично или если позже ставим более тяжёлые винты;
- если будет найден тест с 13–15 inch винтом и током выше 30 A, нужно пересмотреть ESC.

### Если переходить к T-Motor MN3508 KV380 / 15x5.5 / 6S

Нужно найти thrust table именно для этой связки.

Предварительно:

- `40 A` может быть рабочим уровнем;
- если max current приближается к 35–40 A, лучше брать `50 A`;
- если планируется payload и долгие hover-тесты, запас по току и охлаждению важнее экономии.

## Чеклист перед покупкой ESC

1. Поддерживает ли ESC `6S` с полным зарядом 25.2 В?
2. Какой max current у мотора с выбранным винтом по thrust table?
3. Есть ли запас по continuous current хотя бы 25–30%?
4. Подходит ли управляющий сигнал для Pixhawk 6C Mini и ArduPilot?
5. Нужна ли калибровка throttle range?
6. Есть ли BEC, и как он будет подключён или отключён?
7. Хватит ли охлаждения при длительном hover?
8. Удобно ли разместить ESC на лучах рамы?
9. Подходят ли провода, разъёмы и PDB по току?
10. Есть ли реальные отзывы или тесты с похожими моторами и винтами?

## Предварительная рекомендация

Для первого MVP не стоит брать самый дешёвый безымянный ESC.

Базовая логика:

- для Cosmicrc 4108 по текущим данным: `EMAX BLHeli 40A 2-6S` выглядит разумным запасом;
- если разница в цене небольшая, можно взять `EMAX BLHeli 50A 2-6S` как более спокойный запас;
- для более качественной силовой части с 15-inch винтами: `40–50 A 6S ESC`, после проверки thrust table;
- отдельные ESC на лучах предпочтительнее, чем 4-in-1;
- Pixhawk должен питаться через power module, а не через BEC от ESC.

Текущий выбор: `EMAX BLHeli 40A / 50A 2-6S ESC` с AliExpress. Финальную версию 40A или 50A нужно выбрать после проверки thrust table для мотора и винта.
