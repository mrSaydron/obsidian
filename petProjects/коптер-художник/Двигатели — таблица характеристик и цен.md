## Как читать таблицу

AliExpress часто показывает разные цены для разных вариаций товара, количества, склада, купонов и доставки. Поэтому цена в таблице — ориентир на момент извлечения из доступных индексов, а не финальная цена покупки.

Перед покупкой по каждому мотору нужно вручную проверить:

- выбранную вариацию: 1 шт / 2 шт / 4 шт;
- реальную цену с доставкой;
- наличие thrust table;
- совместимость с пропеллером, ESC и батареей;
- массу мотора;
- диаметр вала и крепление пропеллера;
- отзывы с фотографиями.

### Предварительный вывод

`D4108 600KV` намного ближе к нужному направлению, чем ошибочно найденный ранее `D2814 1000KV`: это уже низко-/среднеоборотный мотор под мультикоптеры класса F550/F600/X500, потенциально пригодный для 6S и больших пропеллеров.

Но для проекта «коптер-художник» его нельзя уверенно выбрать без трёх вещей:

1. thrust table именно для `D4108 600KV`;
2. строка для 6S и реального пропеллера, например 12–14 inch;
3. масса мотора и максимальный ток.

Если по таблице он даёт хотя бы `1.5–1.8 кгс` на мотор с нормальным током и hover попадает примерно в `40–60%`, его можно рассматривать как бюджетный кандидат для MVP. Если thrust table нет, риск покупки высокий.

### Предварительный вывод

`Cosmicrc 4108 380KV / 600KV` выглядит как потенциально интересный бюджетный мотор того же общего класса, что и другие `4108`-кандидаты.

Для нашего проекта наиболее интересна версия `380KV`, если она реально рассчитана на `6S` и крупный пропеллер. Версия `600KV` может быть полезна для меньшего пропеллера или 4S/6S-сетапа, но без таблицы тяги нельзя понять, где будет нормальный hover.

По таблице мотор даёт примерно `1.5–1.62 кгс` максимальной тяги на мотор. Для квадрокоптера это около `6.0–6.5 кгс` суммарной статической тяги.

Если целевая взлётная масса MVP будет около `3.0 кг`, thrust-to-weight получится примерно `2.0+`, что уже приемлемо. Если масса уйдёт к `3.5–4.0 кг`, запас станет заметно хуже, особенно для будущего payload.

Главный минус карточки теперь не отсутствие таблицы вообще, а её неполнота. Перед покупкой ещё нужно найти или запросить у продавца:

1. массу мотора;
2. размер вала и крепёжные размеры;
3. подтверждение допустимого напряжения для выбранной версии;
4. таблицу для 380KV на 6S с 13–15 inch пропеллерами, если такие пропеллеры планируются;
5. реальные отзывы/тесты именно этой модели.

Статус: можно оставить в списке кандидатов. Версия `380KV` на `22.2V` выглядит наиболее релевантной для 6S MVP, но запас тяги умеренный, не heavy-lift.
