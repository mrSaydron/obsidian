# Коптер-художник — индекс разработки

Источник: [[Техническое задание]]

## Цель проекта

Разработать мультикоптерную платформу для будущей покраски вертикальных поверхностей: стен, фасадов, промышленных конструкций и труднодоступных объектов.

Главный принцип: не начинать с полноценной покраски. Сначала строится стабильная летающая платформа, затем добавляются удержание у стены, имитация распыления и только после этого реальная покраска.

## Текущий фокус

Фокус текущей разработки — Этап 1: стабильный летающий MVP без покраски, шлангов, payload, RTK и lidar.

Минимальные признаки успешного этапа:

- коптер стабильно взлетает и садится;
- управляется вручную;
- держит позицию;
- передаёт телеметрию;
- выполняет базовые waypoint-миссии;
- имеет понятный запас по массе и питанию для будущих экспериментов.
