# Комплектующие MVP

Источник: [[Техническое задание]]

## Базовая конфигурация из ТЗ

### Рама

Holybro X500 V2

Причины:

- популярная developer platform;
- совместимость с PX4/ArduPilot;
- достаточно места под компоненты;
- подходит для экспериментов.

### Flight Controller

Matek H743-WING V3

Причины:

- STM32H7;
- поддержка ArduPilot/PX4;
- дешевле Pixhawk 6X;
- достаточно интерфейсов UART/CAN;
- подходит для будущей autonomy.

Вопрос для проверки: WING-версия рассчитана на самолёты/VTOL, поэтому перед покупкой нужно проверить удобство подключения именно для multirotor: питание, разъёмы, IMU, mounting, документация ArduPilot.

### Firmware

ArduPilot

Причины:

- зрелая экосистема;
- хорошо подходит для payload drones;
- удобен для MVP;
- хорошая документация.

### Моторы

T-Motor MN3508 KV380

Причины:

- эффективный hover;
- low KV;
- большие пропеллеры;
- низкие вибрации;
- подходят для payload.

### ESC

HobbyWing XRotor Pro 40A

Причины:

- надёжность;
- стабильный throttle;
- подходит для 6S.

### Пропеллеры

15x5.5 carbon props

Причины:

- высокая эффективность;
- низкий шум;
- лучше для hover;
- потенциально меньше агрессивная турбулентность, чем у маленьких скоростных пропеллеров.

### Питание

6S Li-Ion battery

Причины:

- лучше для долгого hover;
- выше энергоёмкость;
- меньше токовые пики по сравнению с агрессивными FPV-сетапами.

## Компоненты, которые нужно добавить в список MVP

В ТЗ указаны основные узлы, но для реальной сборки ещё нужны:

- GPS + compass module;
- power module / current sensor;
- PDB или распределение питания;
- RC transmitter;
- RC receiver;
- telemetry radio или другой канал телеметрии;
- зарядное устройство для 6S;
- кабели, разъёмы, крепёж;
- vibration damping для flight controller;
- landing gear;
- buzzer / finder;
- safety switch, если требуется;
- запасные пропеллеры.

## Что проверить перед закупкой

- Расчёт тяги и веса.
- Совместимость моторов с 15x5.5 props на 6S.
- Ток hover и максимальный ток.
- Запас ESC по току.
- Реальную массу батареи.
- Центр масс с выбранной батареей.
- Габариты пропеллеров на X500 frame.
- Совместимость контроллера с ArduPilot Copter.
- Доступность компонентов в текущем регионе.

## Предварительные конфигурации

### Ultra budget MVP

Цель: минимально проверить полётную платформу.

Особенности:

- бюджетная X500-совместимая рама;
- более доступный flight controller;
- без дорогих RTK/lidar;
- обычный GPS/compass;
- базовая телеметрия.

Риск: меньше запас по качеству и надёжности.

### Balanced MVP

Цель: собрать платформу, которую не придётся сразу переделывать.

Особенности:

- Holybro X500 V2 или аналог хорошего качества;
- надёжные моторы/ESC;
- ArduPilot-совместимый контроллер;
- нормальный GPS/compass;
- телеметрия;
- запас по payload для будущих mock-тестов.

Это рекомендуемый вариант для проекта.

### Future industrial version

Цель: будущая рабочая система покраски.

Особенности:

- hexacopter или X8;
- RTK;
- lidar;
- companion computer;
- наземная подача краски;
- система управления шлангом;
- расширенные failsafe-сценарии.

Этот вариант не стоит собирать первым.
