## Базовая конфигурация из ТЗ

### Рама

Holybro X500 V2

Причины:

- популярная developer platform;
- совместимость с PX4/ArduPilot;
- достаточно места под компоненты;
- подходит для экспериментов.

### Flight Controller

Pixhawk 6C Mini kit

Причины:

- современный Pixhawk-класс на STM32H7;
- хорошая поддержка ArduPilot/PX4;
- проще интегрировать в MVP, чем Matek H743-WING V3;
- обычно продаётся комплектом с GPS/compass, power module, safety switch, buzzer и кабелями;
- подходит для waypoint missions, telemetry и будущей autonomy.

Вопросы для проверки перед покупкой:

- точная комплектация конкретного kit;
- совместимость power module с 6S батареей и ожидаемым током MVP;
- наличие GPS/compass в комплекте;
- схема подключения ELRS receiver;
- достаточно ли UART/CAN/PWM портов для текущей сборки и будущих экспериментов.

### Firmware

ArduPilot Copter

Связанный документ: [[Выбор компонентов/Как выбирать Firmware для MVP|Как выбирать Firmware для MVP]]

Статус: не физическая комплектующая, а выбранная программная основа автопилота.

Кратко: для MVP оставляем ArduPilot Copter, потому что он хорошо подходит для payload-коптеров, waypoint missions, telemetry, onboard logs, failsafe и будущей интеграции через MAVLink.

### Моторы

#### Текущее бюджетное решение

Cosmicrc 4108 380KV

Связанный документ: [[Детали и ссылки/Двигатели — таблица характеристик и цен#Подробности по позиции 2 — Cosmicrc 4108 380KV / 600KV|Двигатели — таблица характеристик и цен]]

Причины:

- заметно дешевле T-Motor MN3508 / MN4014;
- есть базовая load test table из карточки товара;
- версия 380KV тестировалась на 22.2 V, что соответствует 6S nominal;
- по таблице даёт до 1620 г тяги на мотор с APC12x3.8;
- для лёгкого/среднего MVP может дать суммарную статическую тягу около 6.0–6.5 кгс на квадрокоптер.

Ограничения:

- это не лучший мотор по качеству и документации;
- запас тяги умеренный, не heavy-lift;
- масса мотора не найдена и должна быть проверена перед финальным расчётом;
- нужно проверить размер вала, mounting pattern и совместимость с выбранными пропеллерами;
- для будущего payload может быстро стать узким местом.

Предварительная оценка:

- если взлётная масса MVP будет около 3.0 кг, thrust-to-weight может быть около 2.0;
- если масса уйдёт к 3.5–4.0 кг, запас станет заметно хуже;
- для первого доступного MVP мотор можно принять как рабочий бюджетный кандидат, но не как финальное промышленное решение.

#### Качественная альтернатива из исходного ТЗ

T-Motor MN3508 KV380

Причины:

- эффективный hover;
- low KV;
- большие пропеллеры;
- низкие вибрации;
- подходят для payload.

Минус:

- существенно дороже Cosmicrc 4108; на текущем этапе цена может быть неоправданной для первого учебно-инженерного MVP.

### ESC

EMAX BLHeli 40A / 50A 2-6S ESC

Связанный документ: [[Выбор компонентов/Как выбирать ESC для MVP|Как выбирать ESC для MVP]]

Ссылка: https://aliexpress.ru/item/1005009019606904.html

Причины:

- доступен на AliExpress;
- есть варианты 40A / 50A / 60A / 80A;
- версии 40A+ поддерживают 2-6S;
- BLHeli-серия подходит для обычного PWM-управления от Pixhawk/ArduPilot;
- отдельные ESC удобно ставить на лучи X500.

Что проверить перед покупкой:

- выбрать версию не младше 40A;
- если финальный мотор/винт покажет max current ближе к 35-40A, взять 50A;
- точную версию и поддержку 2-6S;
- continuous current и burst current;
- совместимость с Pixhawk 6C Mini / ArduPilot;
- не питать Pixhawk от BEC ESC, а использовать power module;
- запас по току под выбранный мотор, винт и батарею.

### Пропеллеры

15x5.5 carbon props

Причины:

- высокая эффективность;
- низкий шум;
- лучше для hover;
- потенциально меньше агрессивная турбулентность, чем у маленьких скоростных пропеллеров.

### Питание

6S 10000-12000 mAh battery

Связанный документ: [[Выбор компонентов/Как выбирать аккумулятор для MVP|Как выбирать аккумулятор для MVP]]

Предварительный выбор:

- для первых тестов практичнее 6S LiPo, если нужен запас по току;
- 6S Li-Ion рассматривать только после проверки continuous current;
- ёмкость ориентировочно 10000-12000 mAh;
- массу держать примерно в диапазоне 1.2-1.4 кг, пока не обновлён расчёт тяги;
- разъём предпочтительно XT90, если power module и силовая разводка это поддерживают.

Причины:

- 6S снижает ток при той же мощности по сравнению с 4S;
- 10000-12000 mAh даёт разумный компромисс между временем полёта и массой;
- слишком большая батарея может ухудшить hover throttle и съесть payload-запас.

### Управление и связь

RadioMaster Pocket ELRS + ELRS CRSF receiver + telemetry radio

Связанный документ: [[Выбор компонентов/Как выбирать управление и связь для MVP|Как выбирать управление и связь для MVP]]

Предварительный выбор:

- пульт: RadioMaster Pocket ELRS;
- приёмник: ELRS receiver с CRSF по UART, не PWM-only;
- телеметрия: SiK/3DR telemetry radio 433/915 MHz после проверки частоты;
- наземная станция: Mission Planner, QGroundControl как альтернатива;
- режимы на пульте: Stabilize, AltHold/Loiter, RTL.

Причины:

- ELRS даёт хороший запас по качеству RC-связи;
- CRSF по UART проще и аккуратнее, чем PWM-приёмник;
- telemetry нужна для настройки, наблюдения, логов и waypoint missions;
- RTL/failsafe должны быть доступны до первых серьёзных полётов.

### Распределение питания

Power module + PDB / power distribution

Связанный документ: [[Выбор компонентов/Как выбирать PDB и power module для MVP|Как выбирать PDB и power module для MVP]]

Предварительный выбор:

- power module из Pixhawk 6C Mini kit, если он поддерживает 6S и нужный ток;
- PDB отдельная или встроенная в раму, в зависимости от выбранной X500-рамы;
- разъём батареи предпочтительно XT90 / XT90-S, если совместимо с power module;
- Pixhawk питать только через `POWER1`, не через BEC от ESC.

Причины:

- power module нужен для питания Pixhawk и battery monitor;
- PDB нужна для распределения силового тока на 4 ESC;
- токовый запас нужно считать по максимальному току моторов, а не по hover.

## Компоненты, которые нужно добавить в список MVP

В ТЗ указаны основные узлы, но для реальной сборки ещё нужны:

- GPS + compass module;
- зарядное устройство для 6S;
- кабели, разъёмы, крепёж;
- vibration damping для flight controller;
- landing gear;
- buzzer / finder;
- safety switch, если требуется;
- запасные пропеллеры.

## Что проверить перед закупкой

- Расчёт тяги и веса.
- Совместимость моторов с 15x5.5 props на 6S.
- Ток hover и максимальный ток.
- Запас ESC по току.
- Реальную массу батареи.
- Центр масс с выбранной батареей.
- Габариты пропеллеров на X500 frame.
- Совместимость контроллера с ArduPilot Copter.
- Доступность компонентов в текущем регионе.

## Предварительные конфигурации

### Ultra budget MVP

Цель: минимально проверить полётную платформу.

Особенности:

- бюджетная X500-совместимая рама;
- более доступный flight controller;
- без дорогих RTK/lidar;
- обычный GPS/compass;
- базовая телеметрия.

Риск: меньше запас по качеству и надёжности.

### Balanced MVP

Цель: собрать платформу, которую не придётся сразу переделывать.

Особенности:

- Holybro X500 V2 или аналог хорошего качества;
- надёжные моторы/ESC;
- ArduPilot-совместимый контроллер;
- нормальный GPS/compass;
- телеметрия;
- запас по payload для будущих mock-тестов.

Это рекомендуемый вариант для проекта.

### Future industrial version

Цель: будущая рабочая система покраски.

Особенности:

- hexacopter или X8;
- RTK;
- lidar;
- companion computer;
- наземная подача краски;
- система управления шлангом;
- расширенные failsafe-сценарии.

Этот вариант не стоит собирать первым.
