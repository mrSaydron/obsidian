# Расчёт веса и тяги MVP

Источник: [[План разработки MVP]], [[Комплектующие MVP]], [[Сметы MVP — варианты комплектующих AliExpress]]

Дата: 2026-05-15

## Цель

Проверить, какой из вариантов MVP имеет достаточный запас тяги для первого стабильного летающего макета.

Ключевой критерий для первого MVP:

- минимально приемлемо: thrust-to-weight ratio около 1.5;
- комфортно для настройки и будущих экспериментов: 2.0+;
- желательный hover throttle: примерно 40–55%;
- рискованная зона: hover throttle выше 65–70%, потому что остаётся мало запаса на ветер, манёвры, просадку батареи и будущий payload.

## Важные допущения

Это предварительный инженерный расчёт, а не финальный eCalc/стендовый тест.

Перед покупкой нужно заменить допущения на реальные данные из карточек товаров/даташитов:

- масса рамы/ARF-комплекта;
- масса каждого мотора и ESC;
- реальная масса батареи;
- максимальная тяга мотора с конкретным пропеллером и напряжением;
- рекомендуемый диапазон пропеллеров для мотора;
- допустимый ток ESC;
- совместимость пропеллеров с рамой.

## Вариант A — Balanced X500 + Pixhawk 6C Mini kit + X500 ARF силовая часть

Связанная смета: [[Сметы MVP — варианты комплектующих AliExpress#Вариант 3 — Balanced Holybro X500 + Pixhawk 6C Mini kit]]

### Принятые допущения

| Параметр | Значение |
|---|---:|
| Сухая масса: рама/силовая часть/FC/GPS/радио/мелочёвка | 1.65 кг |
| Батарея 6S 10000–12000 mAh | 1.25 кг |
| Оценочная взлётная масса | 2.90 кг |
| Оценочная максимальная тяга на мотор | 0.95–1.15 кгс |
| Оценочная суммарная тяга | 3.80–4.60 кгс |

### Результат

| Метрика | Значение |
|---|---:|
| Thrust-to-weight ratio | 1.31–1.59 |
| Оценочный hover throttle | 63–76% |

### Вывод

Этот вариант выглядит удобным по интеграции ArduPilot/Pixhawk, но по силовой части может оказаться на границе.

Главный риск: если X500 ARF использует моторы/пропеллеры, рассчитанные скорее на лёгкий 4S/10-inch setup, то с тяжёлой 6S 10000–12000 mAh батареей коптер будет летать, но без комфортного запаса.

Перед покупкой обязательно проверить:

- какие именно моторы входят в X500 V2 ARF/power kit;
- допустимое напряжение этих моторов/ESC;
- размер пропеллеров в комплекте;
- thrust table для выбранной пары мотор + пропеллер + батарея;
- рекомендуемую батарею для X500 ARF от производителя.

Предварительная оценка: годится как учебный MVP, но может быть слабой базой для будущего payload/wall-following.

## Вариант B — High-quality X500 + T-Motor MN3508 KV380 + 15x5.5 + 6S

Связанная смета: [[Сметы MVP — варианты комплектующих AliExpress#Вариант 4 — High-quality original-TZ components]]

### Принятые допущения

| Параметр | Значение |
|---|---:|
| Сухая масса: рама/моторы/ESC/FC/GPS/радио/мелочёвка | 2.15 кг |
| Батарея 6S 10000–12000 mAh | 1.25 кг |
| Оценочная взлётная масса | 3.40 кг |
| Оценочная максимальная тяга на мотор | 1.75–2.15 кгс |
| Оценочная суммарная тяга | 7.00–8.60 кгс |

### Результат

| Метрика | Значение |
|---|---:|
| Thrust-to-weight ratio | 2.06–2.53 |
| Оценочный hover throttle | 40–49% |

### Вывод

Этот вариант намного лучше по запасу тяги и больше похож на платформу, которую можно развивать дальше.

Минусы:

- почти вдвое дороже Balanced X500/Pixhawk 6C Mini по смете;
- выше цена ошибки при сборке и настройке;
- нужно отдельно подтвердить, что 15x5.5 пропеллеры физически нормально встают на выбранную раму X500;
- Matek H743-WING V3 всё ещё требует проверки именно для ArduPilot Copter, если оставлять его в этой конфигурации.

## Сравнение

| Критерий | Balanced X500 + Pixhawk 6C Mini | X500 + MN3508 + 15x5.5 |
|---|---:|---:|
| Оценочная взлётная масса | 2.90 кг | 3.40 кг |
| Суммарная тяга | 3.80–4.60 кгс | 7.00–8.60 кгс |
| Thrust-to-weight | 1.31–1.59 | 2.06–2.53 |
| Hover throttle | 63–76% | 40–49% |
| Интеграционный риск | ниже по FC, выше по силовой части | выше по подбору, ниже по запасу тяги |
| Потенциал под payload | низкий/средний | средний/хороший |
| Стоимость | ниже | выше |

## Предварительная рекомендация

Не покупать Balanced X500 + Pixhawk 6C Mini вслепую, пока не подтверждена реальная силовая часть X500 ARF.

Если цель — просто научиться ArduPilot и получить первый полёт дешевле, Balanced-вариант допустим.

Если цель — именно база для проекта «коптер-художник», лучше двигаться к конфигурации с большим low-KV мотором и 15-inch пропеллерами, либо искать промежуточный вариант:

- оставить Pixhawk 6C Mini kit как более простой flight controller;
- заменить/выбрать силовую часть с запасом тяги ближе к варианту MN3508;
- не брать слишком тяжёлую батарею до проверки thrust table.

## Следующие действия перед покупкой

1. Проверить точную комплектацию Holybro X500 V2 ARF / power kit.
2. Найти thrust table для моторов из X500 ARF.
3. Найти thrust table для T-Motor MN3508 KV380 + 15x5.5 + 6S.
4. Проверить, влезают ли 15-inch пропеллеры на X500 без пересечения дисков и без опасного зазора.
5. Выбрать батарею не по ёмкости, а по компромиссу масса/ток/время hover.
6. После уточнения реальных масс пересчитать таблицу.

## Решение на текущий момент

Текущий лучший путь: не финализировать покупку, а сделать короткий compatibility-check по силовой части.

Главный вопрос: покупать готовый X500 ARF как учебный стенд или сразу собирать более тяговитую силовую часть под будущие эксперименты.
