Разработать андроид приложение для игры в мафию где телефон выступает в качестве ведущего
Разработку планирую вести совместно с chatGPT

Приложение должно уметь регистрировать игроков, выбирать для них роли.
Во время игры приложение должно озвучивать этап игры, и выводить интерфейс для роли которая играет в текущий момент.
В момент общего совещания, программа должна показывать результаты за ночь и давать возможность голосования.
