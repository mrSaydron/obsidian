Новая попытка реализации приложения для изучению английского языка.
В качестве названия, возможно не окончательное, решил выбрать слово из санскрита: tala. Которое обозначает "ритм", что должно подчеркивать регулярные занятия языком. А короткое и необычное название должно выделять это приложение

Ставлю основной целью в которой должно помогать данное приложение, это изучение слов с помощью перевода предложений. Вторая цель, это изучение правил. Метод изучения, регулярное повторение в виде карточек. С алгоритмом прогресса аналогичным тому, который используется в приложении anki.

Для генерации приложений и проверки правильности написания хочу попробовать одну из доступных LLM моделей. От которой будет требоваться следующий функционал:
- генерация предложений для изучения основанных на словаре пользователя, с включением изучаемого слова
- генерация предложений с знакомыми для пользователя грамматическими правилами
- генерация предложений для изучения грамматических правил
- проверка предложения введенного пользователем
- предложения лучшего перевода, если такой имеется
- указание на статью с правилом, если в переводе имеется ошибка

Приложение должно работать на платформе android с как можно меньшей зависимостью, а в идеале с отсутствием, бэкенда
