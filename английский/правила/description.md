# Правила английского языка

Раздел для интервального изучения правил английского языка.

## Как работаем

1. Пользователь сообщает, что прошел новую тему.
2. Для темы создается отдельный файл в этом разделе. Имя файла совпадает с названием темы, например `Present Perfect.md`.
3. В файл темы добавляются:
   - краткое описание правила;
   - что именно изучаем;
   - примеры;
   - задания;
   - история вопросов, ответов, ошибок и разбора.
4. В `lessons.md` добавляется или обновляется запись по теме:
   - ссылка на файл темы;
   - дата следующего повторения;
   - параметры SuperMemo SM-2: repetitions, interval, ease_factor;
   - последняя оценка ответа.
5. Один раз в день выбирается не больше одной темы, у которой `next_review` меньше или равен текущей дате.
6. После ответа пользователя тема оценивается, ошибки разбираются, а дата следующего повторения пересчитывается по SM-2.

## Шкала оценки SM-2

- 5 — идеально, без ошибок.
- 4 — правильно, но были небольшие сомнения или мелкие недочеты.
- 3 — в целом вспомнил, но были заметные ошибки.
- 2 — вспомнил частично, нужно повторить скоро.
- 1 — почти не вспомнил.
- 0 — не вспомнил совсем.

Для оценок ниже 3 серия повторений сбрасывается, а тема ставится на скорое повторение.
