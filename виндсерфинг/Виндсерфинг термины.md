Направление ветра
    фордевинд - впрямо по ветру (ветер дует в корму)
    бакштаг - ветер сзади сбоку
    - полный бакштаг - больше сзади
    - крутой бакштаг - больше сбоку
    галфвинд - ветер сбоку
    - левый галс - ветер в левый борт
    - правый галс - ветер в правый борт
    бейдевинд - ветер спереди сбоку
    оверштаг(левентиг) - против ветра

Повороты
    увалиться - повернуться по ветру
    привестись - поворачиваться против ветра
    оверштаг - поворот при котором нос лодки повернут против ветра (парус проходит через корму)
    фордевинд - поворот при котором нос лодки повернут по ветру (парус проходит через нос)
