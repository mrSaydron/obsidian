синтаксис установки переменной **оболочки** (локальная переменная) для текущего терминала
    {var_name}={value}

синтаксис для установки переменной окружения для одной команды
    {var_name}={value} {command}
    HOME=/tmp cd - перейдет в папку /tmp так как для текущей команды будет изменена домашняя дирректория

синтаксис для установки переменной окружения (будет доступна для всех дочерних приложений, глобальная переменная) для сессии
    export {var_name}={value}

$HOME - домашняя директория пользователя
