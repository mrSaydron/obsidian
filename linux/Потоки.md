
Типы потоков:
- STDIN - Standart Input - 0
- STDOUT - Standart Output - 1
- STDERR - Standart Error - 2

**>** - переправляет поток STDOUT в файл, перезаписывая его если он уже существует
    ls -la > output - записывает результаты работы приложения ls в файл output

**>>** - перенаправляет поток STDOUT в файл, дописывая его
    echo 'hello' >> result - добавляет строчку hello в файл

**<** - перенаправляет поток STDIN
    ws -l < result - перенаправляет входной поток из файла в приложение ws
    ws -l < result > output - перенаправляет входной поток из файла в приложение ws и сохраняет результат в файле output

##### Указание типа потока при перенаправлении

**&** - ставится перед номером потока куда будет перенаправление
    ls lala > output 2>&1 - перенаправляет потоки STDOUT в файл, а поток STDERR в поток STDOUT 

**2>** - перенаправляет конкретный (2) поток в файл
    cd lala 2> output - перенаправляет только поток STDERR в файл

**&>** - перенаправление обоих потоков STDOUT и STDERR
    cd lala &> output - перенаправляет оба потока в файл

Символы > < и >> предназначены для связывания входного и выходных потоков с файлами, но не между программами

**|** - связывает выходной и входной потоки между двумя программами (называется пайплайном)
    cat .bashrc | grep alias | grep color
