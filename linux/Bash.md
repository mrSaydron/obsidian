### Заголовок bash скрипта:
\#!/bin/bash

### Расширение скрипта:
Расширение не требуется, но обычно используют sh

### Запуск скрипта:
./<имя скрипта>
bash <имя скрипта>

### Переменные:
Запись (перезапись) значения:
<имя>=<значение>

Использование переменной:
$<имя>
${<имя>}
Если переменная окажется между одинарных кавычек, то она будет восприниматься как строка.
Если переменная окажется между двойных кавычек, то она будет восприниматься как переменная.

При обращении к необъявленной переменной возвращается пустая строка

### Передача аргументов в скрипт
пример: ./script.sh arg1 arg2 arg3

для получения аргументов используются специальные переменные:
$1 - первый аргумент
$2 - второй аргумент
....
$0 - имя скрипта
$# - количество аргументов

### Ветвление:
```
if [[ условие ]]; then   
    # дейсвия, если условие истинно  
fi
```

```
if [[ условие ]]; then
	# дейсвие, если условие истино
else
	# действие, если условие не истино
fi
```

```
if [[ условие ]]; then
	# действие
elif [[ еще одно уловие ]]; then
	# действие
else
	# действие
fi
```

**case синтаксис:**
```
case переменная in
значение1)
	# действие
	;;
значение2)
	# действие
	;;
*)
	# действие, если переменная не равно ни одному из вариантов
esac
```

**Условия для строк:**  
	-z \<строка> # строка пустая  
	-n \<строка> # строка не пустая  
	\<стр1> == \<стр2> # строки равны  
	\<стр1> != \<стр2> # строки не равны
	\<стр1> \< \<стр2> # меньше
	<стр1> \> \<стр2> # больше

**Условия для чисел:**
	-eq, # равно
	-ne, # не равно
	-lt, # меньше
	-le, # меньше или равно
	-gt, # больше
	-ge, # больше или равно

**Условия для файлов:**
	-e <путь> # путь существует
	-f <путь> # это файл
	-d <путь> # это директория
	-s <путь> # размер файла больше 0
	-x <путь> # файл исполняемый

**Условия логические:**
	! # логическое НЕ
	&& # логическое И
	|| # логическое ИЛИ

### Циклы:
```
for переменная in список_значений
do
	# действие
done
```

в качестве разделителя для значений используется пробел

```
while [[ условие ]]
do
	# действие
done
```

для прерывания цикла можно использовать: break, continue

### Команды:
**read** <переменная> - ожидает ввод от пользователя и записывает его в переменную

### Арифметика
```
let "переменная=выражение"
```

Пример:
```
let "c=1+1"
let "c=a+b"
```

### Запуск приложений
```
результат=`программа передаваемые_параметры`

пример:

a=`echo "test"`
files=`ls ~`
```

результат работы программы
0 - корректное завершение
не 0 - код ошибки

скрипт так же может возвращать результат:
```
exit код
```

узнать код только что завершившейся программы:
```
echo $?
```

так как условие if принимает ????
0 - true
!0 - false

### Функции

```
имя_функции ()
{
  # действия с $1, $2 ... $#(количество аргументов)
}

имя_функции аргумент1 аргумент2 ...
```

видимость переменных
```
имя_функции ()
{
  var_global=1
  local var_local=1
}

имя_функции
echo $var_global # выведет 1
echo $var_local # ничего не выведет
```

Компактная запись
```
имя_функции () { действие1; действие2; }
```
