[FHS - стандарт иерархии файловой системы](https://ru.wikipedia.org/wiki/FHS)

**/bin** - основные утилиты
**/etc** - (et cetera) содержит конфигурацию программ в обычных текстовых файлах
**/home** - содержит домашние директории пользователей системы, за исключением суперпользователя (root)
**/lib** - основные библиотеки
**/mnt** - временно монтируемые файловые системы
**/opt** - дополнительные программное обеспечение
**/root** - домашний каталог пользователя root
**/sys** - информация об устройствах, драйверах
**/tmp** - временные файлы
**/usr** - вторичная иерархия для данных пользователя
**/var** - изменяемые файлы
**/var/log** - каталог для логов
