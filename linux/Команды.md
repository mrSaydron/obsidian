**pwd** - (print working directory) выводит текущую директорию

**ls** - (list) отображает файлы в текущей или переданной директории
    -a отображать скрытые файлы
    -l отображение в виде списка

**cd** - (change directory) перемещается в указанную директорию

**man** - выводит информацию по команде

**stat** - (file system status) возвращает статус для файла или директории

**cat** - выводит на экран содержимое файла

**head** - показывает первые 10 строк файла, если не указано другое значение
    -n количество строк которое необходимо вывести

**tail** - показывает последний 10 строк файла, если не указано другое значение
    -n количество строк которое необходимо вывести
    -f продолжает писать выводить новые строки на консоль

**less** - "пейджер" приложение для просмотра файлов. Хорошо подходит для просмотра файлов большого размера
    Команды:
    q - выход
    f - вперед на страницу
    b - назад на страницу
    /{text} - поиск текста
    n - следующее совпадение
    N - предыдущее совпадение

**gerp** - выводит на экран только строки в которых находится искомая фраза в указаном файле или через пайп
    -B --before-context - количество строк выводимых до искомой строки
    -A --after-context - количество строк выводимых после искомой строки
    -C --context - количество строк до и после искомой строки
    -r - указывает, что искать надо не в конкретном файле, а в директории
    -n - указывает номер строки в котором была найдена искомая фраза
    -c - подсчитывает количество раз встречалась искомая фраза

**sed** - потоковый текстовый редакто. Позволяет редактировать текст с помошью команд

**ws** - подсчитывает количество слов или строк
    -l - указывает, что надо считать не слова а строки

**uniq** - выводит только уникальные строки

**sort** - выводит отсортированные строки

**echo** - выводит на экран переданное ей значение, можно использовать для вывода переменных окружения

**export** - сохраняет переменную как переменную окружения
    -f - сохраняет функцию
    -n - удаляет перемнную из окружения
    -p - выводит список всех экспортированных перемнных

**history** - показывает историю

**whoami** - печатает логин текущего пользователя

**ps** - выводит запущенные процессы (без параметров только для текущего терминала)
    aux - выведет информацию от имени какого пользователя был запущен процесс
    -e - все запущенные процессы в системе
    -f - дополнительная информация о процессе
    -С - фильтрация по имени (не заработала)
    -p - поиск по номеру процесса
    -u - поиск по имени пользователя
    
    https://www.baeldung.com/linux/ps-command

**id** - выводит группы в которых состоит пользователь

**su** - (switch user) переключает пользователя

**sudo** - (substitute user and do) подменяет пользователя для выполнения
    -u - указывает под каким пользователем должна выполняться команда (по умолчанию root)
    -i - запускает новую оболочку внутри текущей от необходимого пользователя

**chown \<user\> \<file\>**  - позволяет изменить владельца файла

**chmod \<mod\> \<file\>** - изменить права доступа для файла

**apt** - пакетный менеджер для работы с приложениями
    install - для установки приложений
    remove - для удаления приложений
    update - синхронизирует локальный список доступных приложений с удаленным репозиторием

**type / which / whereis** - выводит путь до приложения

**touch** - изменяет дату последнего доступа к файлу. Если файла нет, создает его

**rm** - удаляет файл
    -r - для удаления папки
    -f / --force - удаляет все файл с ограниченными правами без диалога

**mv** - переименовывает или перемещает файл

**cp** - копирует файл или папку
    -r - для копирования директории

**mkdir** - создает новую папку
    -p / --parents - создает папку со всеми подпапками

**env** - выводит переменные окружения

**wget** - делает запрос по переданной ссылке и сохраняет результат в файл
    -r - рекурсивная загрузка файлов со страницы
    -l - глубина загрузки
    -A - указание типов файлов для загрузки через запятую

**unzip** - распаковывает архив с типом zip

**gunzip** - распаковывает архив с типом gz, удаляя его после удаления

**bunzip2** - распаковывает архив с типом bz2 

**zip** - запаковывает указанные файлы в zip архив

**gzip** - запаковывает указанный файл (один) в gz архив и удаляет файлы. При этом упаковать может только один файл

**bzip2** - запаковывает файлы в bz2 архив

**tar** - запаковывает указанные файлы в tar архив
    -cvf - упаковывает без сжатия
    -zcvf - упаковывает с сжатием используя gzip
    -xvf - распаковать архив
    -xzvf - распаковать архив с использованием gunzip
    -cjvf - упаковывает используя bzip2
    -xjvf - распаковывает используя bzip2

**find** - ищет файл в указаной директории
    find <папка> -name <файл>
    find directory criteria action
    **criteria:**
    -name - ищет по имени файла
    -iname - ищет по имени файла не зависимо от регистра
    -type - ищет по типу файла: f - простые файлы, d - каталоги, l - символьные ссылки...
    -size - поиск по размеру файлов "+" - размер больше, "-" - размер меньше
    -empty - ищет пустые файлы и каталоги
    -cmin - поиск по времени изменения
    -atime - поиск по времени доступа
    -perm - поиск по набору разрешений
    -and -or -not - операнды объединения условий
    -user - поиск по пользователю
    **action:**
    -delete - удаляет файл
    -ls - выводит информацию о найденных файлах
    -print - выводит путь до файла (по умолчанию)
    -exec - выполняет команду над каждым файлом
    [# Команда find в Linux – мощный инструмент сисадмина](https://habr.com/ru/companies/alexhost/articles/525394/)

**ssh-keygen** - генерирование ssh ключа для подключения к удаленному серверу

**ssh_add** - добавляет ключ в систему

**scp** - копирует файлы с удаленного сервера или обратно
    scp -P порт логин@адрес_сервера:путь_откуда путь_куда
    scp -P порт путь_откуда логин@адрес_сервера:путь_куда

**apt-get** - пакетный менеджер, устанавливает и удаляет приложения
    install - устанавливает приложение
    remove - удаляет приложение
    update - обновляет ссылки на пакеты
    upgrade - обновление установленных приложений

**free** - отображает сколько памяти в наличии на устройстве

**nproc** - отображает количество ядер на устройстве

**lscpu** - отображает детальную информацию о проецессоре

**exit** - выходит из shell или из скрипта
