##### В консоли
Ctrl + A - перейти в начало
Ctrl + E - перейти в конец
Ctrl + Shift + C - копировать
Ctrl + Shift + V - вставить
Tab - предложить возможные варианты
Tab + Tab - выбрать первый вариант их предложенных
Ctrl + C - прервать выполнение
Ctrl + Z - приостановить выполнение
    fg - вывод приложения из приостановления
    bg - перевод приложения в фоновый режим

##### less
q - выход
g - в начало файла
/ - поиск по файлу
G - в конец файла

##### nano
Ctrl + x - выход
