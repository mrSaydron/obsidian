Позволяет запрашивать, визуализировать, оповещать и исследовать метрики, журналы и трассировки. Работает с данными из базы основанных на временных рядах, SQL и NoSQL, инструментами CI/CD

Grafana с открытым исходным кодом — это программное обеспечение с открытым исходным кодом для визуализации и аналитики. Оно позволяет вам запрашивать, визуализировать, оповещать и исследовать ваши метрики, журналы и трассировки, независимо от того, где они хранятся. Оно предоставляет вам инструменты для превращения данных вашей базы данных временных рядов (TSDB) в информативные графики и визуализации.

Prometheus - это технология, которая собирает и хранит данные в структуре временных рядов. Имеет свой язык запросов для работы с данными который называется PromQL.
Помогает получать данные от программного обеспечения и ифраструктуры и сохранять их.
Программное обеспечение должно быть оснащено клиентской библиотекой к которой периодический обращается прометеус для сбора метрик
Структура данных для ответа клиентской части прометеуса следующая:
    HELP - \<наименование метрики\> \<описание\>
    TYPE - \<наименование метрики\> \<тип\>
    данные

Телеметрия - процесс передачи и записи показаний приложения

Панель инструментов Grafana состоит из панелей, отображающих данные, диаграмм и других визуализациях

Источником данных - может быть как БД так и другие инструменты Grafana, API возвращающий JSON и даже CSV (всего около 155 вариантов)
Плагин - согласует различия между моделью источника данных и моделью данных в Grafana
Запрос - преобразует полученные данные в данные используемые для визуализации. Можно использовать различные варианты агрегаций. Язык запроса может быть свой собственный для различных типов данных
Трансформация - позволяет формат данных не соответствует можно дополнительно применить трансформацию (на отдельной панели)
Панель - контейнер, который отображает визуализацию и предоставляет инструменты управления и обработки визуализации

Агрегации - объединене набора данных
    Среднее
    min max
    sum
    count

База данных временных рядов (TSDB) - это база данных, специально разработанная для данных временных рядов
- данные добавляются и очень редко изменяются
- хранение не значений а их дельт
- хранение дельт от дельт
- пропуск дельт равных нолю
Примеры TSDB баз данных:
- grafit
- influsDB
- prometheus
